# sysdevex
This is a project of sysdevex.  
`git status`で現在のブランチを確認。  
'git checkout ブランチ名'でブランチの切り替え。  
`git checkout -b 新しいブランチ名`で新しいブランチを作成して切り替え。  

## 基本的な流れ  
1. コードの編集
2. 変更したファイルをステージ
3. メッセージを記入(変更はfix ○○, 追加はadd ○○, 削除はdelete ○○)
4. コミット(まだ包んだだけ)
5. プッシュ(ブランチが正しいか確認！)
6. プルリクエストの作成(GitHubブラウザ上)
